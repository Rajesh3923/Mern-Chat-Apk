## Getting Started

Follow the instructions below to set up and run the Chat App locally on your machine.

### Prerequisites

- Node.js (v16 or higher)
- npm (Node Package Manager)


### Setup .env file

```js
PORT=...
MONGO_DB_URI=...
JWT_SECRET=...
NODE_ENV=...
```

### Build the app

```shell
npm run build
```

### Start the app

```shell
npm start
```
